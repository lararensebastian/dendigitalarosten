Den Digitala Rösten

Ett interaktivt skrivprojekt om AI, källkritik och manipulation.

Eleven hamnar i kontakt med en person online – som visar sig vara ett AI-genererat påverkanskonto. Genom val och skrivuppgifter tränas både svenska och digital kompetens.

🎯 Syfte

Projektet kombinerar:

Svenska (alla centrala texttyper)

Källkritik

Digital kompetens

Demokrati och påverkan

AI och desinformation

Eleverna får uppleva hur manipulation kan byggas upp steg för steg – istället för att bara läsa om det.

🧠 Vad tränas?

Under 9 kapitel tränar eleverna:

Beskrivande text

Berättande text

Argumenterande text

Instruerande text

Reflekterande text

Dialog

Fördjupad källkritisk analys

Varje kapitel kräver minst 100 ord och bygger vidare på elevens tidigare val.

🎮 Hur det fungerar

Eleven gör ett val i varje kapitel

Valen bygger en profil: 🔥 Risktagare / 🔍 Analytiker / 🛡️ Försiktig

Efter valet skriver eleven sin text

Progress sparas lokalt i webbläsaren

Hela boken kan laddas ner som textfil

👩‍🏫 Lärarläge

Aktiveras genom att lägga till:

?teacher=1


i URL:en.

Exempel:

index.html?teacher=1


Då visas bedömningsstöd och lärarnotiser.

🚀 Publicering

Projektet är en statisk HTML-fil.

För GitHub Pages:

Döp filen till index.html

Lägg den i repo-root

Aktivera GitHub Pages under Settings → Pages

Välj branch main och root /

Klart.

⚠️ Viktigt att veta

Progress sparas i LocalStorage

Byter eleven dator/webbläsare försvinner sparad text

Ingen server eller inloggning krävs

📌 Målgrupp

Åk 8–9, IM eller gymnasiet
Passar särskilt bra vid arbete med:

Källkritik

Demokrati

AI och påverkan

Digital säkerhet

💡 Bakgrund

Påverkanskampanjer med AI-genererade profiler är verkliga och pågår just nu i sociala medier.
Detta projekt gör fenomenet konkret – genom att låta eleverna uppleva det inifrån.
